# [Permutation in String](https://leetcode.com/problems/permutation-in-string/)

## Problem Statement:

```
Given two strings s1 and s2, return true if s2 contains a permutation of s1, or false otherwise.

In other words, return true if one of s1's permutations is the substring of s2.
```

## Examples:

```
Example 1:

Input: s1 = "ab", s2 = "eidbaooo"
Output: true
Explanation: s2 contains one permutation of s1 ("ba").

Example 2:

Input: s1 = "ab", s2 = "eidboaoo"
Output: false
```

## Constraints:

```
1 <= s1.length, s2.length <= 1e4
s1 and s2 consist of lowercase English letters.
```

// optimized is that we dont need to compare the two hashmaps every time
// hence reducing complexity from 
     // O(l_1 + 26*(l_2 - l_1)) ==> O(l_1 + (l_2 - l_1)) 
     // where "l_1" is the length of string "s1" and "l_2" is the length of string "s2

class Solution {
public:
    bool checkInclusion(string s1, string s2) {
        int n = s1.size();
        int m = s2.size();
        
        if(n > m) return 0;
        
        int mp[26];
        int cur[26];
        memset(mp, 0, sizeof mp);
        memset(cur, 0, sizeof cur);
        
        int tot = 0;
        for(int i = 0; i < n; i++) {
            mp[s1[i] - 'a']++;
            if(mp[s1[i] - 'a'] == 1) tot++;
        }
        int i = 0;
        int chrs = 0;
        
        for(i; i < n; i++) {
            int ch = s2[i] - 'a';
            cur[ch]++;
            if(cur[ch] == mp[ch]){
                chrs++;
            }
        }
        if(chrs == tot) return 1;
        
        for(i; i < m; i++) {
            int ch = s2[i] - 'a';
            cur[ch]++;
            if(cur[ch] == mp[ch]) chrs++;
            
            ch = s2[i-n] - 'a';
            cur[ch]--;
            if(cur[ch] == mp[ch] - 1) {
                chrs--;
            }
            if(chrs == tot) return 1;
        }
        
        return 0;
    }
};
